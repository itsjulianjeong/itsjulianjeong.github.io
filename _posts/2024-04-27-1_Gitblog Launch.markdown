# 1. 깃허브 블로그를 위한 Repository 만들기

*반드시!* Repository 이름으로 ***본인의 username***.github.io 를 입력한다. _(e.g., itsjulianjeong.github.io)_  
추가로 *Add a README file*도 체크한다.
